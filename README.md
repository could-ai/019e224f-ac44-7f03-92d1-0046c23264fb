# Mass Spectrometry Presentation

Ek interactive Flutter application jo Mass Spectrometry par presentation (PPT) dikhati hai. Isme aap slides ke through navigate kar sakte hain aur basic concepts seekh sakte hain.

## Features
- **Interactive Slides**: Swipe ya buttons ka use karke slides change karein.
- **Mass Spectrometry Topics**: 
  - Introduction to Mass Spectrometry
  - Principle
  - Instrumentation (Ion Source, Mass Analyzer, Detector)
  - Applications
- **Responsive Design**: Mobile aur Web dono par badiya chalta hai.

## Tech Stack
- Framework: Flutter
- Language: Dart
- State Management: Stateful Widgets with PageController

## Setup & Run
1. Ensure Flutter SDK is installed.
2. Run `flutter pub get` to install dependencies.
3. Run the app using `flutter run` on your preferred device.
